DataOps Learning Guide

Module 1: Data Pipeline Architecture
Learners should understand raw bronze silver and gold layers.

Module 2: Metadata Driven Configuration
Learners should understand how pipeline behavior can be controlled using JSON metadata.

Module 3: Load Types
Learners should understand FULL APPEND INCREMENTAL and UPSERT load patterns.

Module 4: Trigger and Scheduling
Learners should understand daily weekly monthly and on demand triggers.

Module 5: Troubleshooting
Learners should understand common pipeline failures such as missing keys invalid watermark columns schema mismatch and failed merges.
